# ExtraDocResponse

### Propriedades:

|                |        |                                        |
| -------------- | :----: | -------------------------------------- |
| open\_id       | String |                                        |
| token          | String | token do documento extra               |
| name           | String | nome do documento extra                |
| original\_file | String | url do documento extra sem assinaturas |
| signed\_file   | String | url do documento extra com assinaturas |
