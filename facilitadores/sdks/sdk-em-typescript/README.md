# SDK em Typescript

O repositório desse SDK pode ser encontrado [aqui](https://github.com/ZapSign/zapsign_SdkNode). Nele é possível ver todo código necessário usado.

Em seu projeto NPM - Typescript/ Node, adicione a [dependência da ZapSign](https://www.npmjs.com/package/sdk-node-typescript-zapsign):

```
npm i sdk-node-typescript-zapsign
```

Agora apenas atualize suas dependências e realize suas requisições.

### Requisições

As requisições feitas pelo SDK são separadas em duas partes:

* [Requisições para documentos](https://docs.zapsign.com.br/facilitadores/sdks/sdk-em-typescript/requisicoes-para-documentos)
* [Requisições para signatários](https://docs.zapsign.com.br/facilitadores/sdks/sdk-em-typescript/requisicoes-para-documentos)
